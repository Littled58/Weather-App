# Weather Forecast App

This is a simple and beautiful weather forecasting web app built using **Streamlit** and **WeatherAPI**. It gives you a 5-day forecast, temperature trends, rain alerts, sunrise/sunset info, and real-time air quality metrics like CO, PM2.5, PM10, and NO2 â€” all in one click.

---

## Features

- 5-day weather forecast
- Temperature trend plot using Matplotlib
- Sunrise & Sunset timings
- Rain alert: Shows if rain is expected
- Air Quality Index (CO, NO2, PM2.5, PM10)
- Smart recommendation: Whether it's a good day to go out!

---

## Technologies Used

- **Python**
- **Streamlit** (UI)
- **WeatherAPI** (data source)
- **Pandas** (data processing)
- **Matplotlib** (graph plotting)
